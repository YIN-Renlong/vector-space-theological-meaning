# CTSB-100 v1 Draft Archive

This folder preserves the first full CTSB-100 draft run.

## Why this archive exists

Version 1 was the first successful 100-concept benchmark run. It used two query conditions:

1. neutral/academic query wording;
2. explicit theological/Catholic query wording.

It produced a valid methodological starting point and demonstrated that:

- the Azure OpenAI embedding pipeline works;
- concept-level Catholic Alignment Scores can be calculated;
- theological context shifts can be measured statistically;
- the benchmark should be refined before final dissertation claims.

## Status

This version is archived as an iterative research stage, not as the final benchmark.

The next version, CTSB-100 v2, adds four query contexts:

1. bare/minimal;
2. ordinary lived usage;
3. neutral academic;
4. explicit Catholic/theological.

It also adds hidden raw CSV export blocks in the dashboard for easier analysis and sharing.
